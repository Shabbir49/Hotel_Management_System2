#  Hotel Management System (Spring Boot)

A **Hotel Management System** built using **Spring Boot, Spring Data JPA, Hibernate, and MySQL**.  
This project performs complete **CRUD operations** for Hotels, Rooms, and Bookings and is designed with clean architecture suitable for **resume and interviews**.

---

##  Features

- Hotel Management (Add, View, Update, Delete)
- Room Management linked with Hotels
- Booking Management linked with Rooms
- Complete CRUD Operations
- RESTful APIs
- JPA Entity Relationships
  - OneToMany
  - ManyToOne
- MySQL Database Integration
- Hibernate ORM
- Clean layered architecture (Controller, Service, Repository)

---

##  Technologies Used

- **Java**
- **Spring Boot**
- **MySQL**
- **REST API**
